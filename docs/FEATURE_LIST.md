# 工程师端小程序功能清单

## 📱 系统概述

工程师端小程序是一个专为IT运维工程师设计的移动工作平台，提供工单管理、耗材管理、团队协作等核心功能。

## 👥 用户角色

### 工程师（Engineer）
- 处理分配的工单
- 记录耗材使用
- 发起求助请求
- 查看个人工作统计

### 经理（Manager）
- 拥有工程师的所有权限
- 分配和管理工单
- 管理耗材库存
- 查看团队统计报表
